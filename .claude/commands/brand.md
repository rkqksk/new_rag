# /brand - Brand Guidelines Management

Create and maintain brand guidelines, style guides, and design systems.

## Usage
```
/brand create guidelines --company "StartupX" --industry tech
/brand analyze logo.png --extract-colors --generate-palette
/brand check design.png --validate-against guidelines.json
```

## Features
- Color palette generation
- Typography guidelines
- Logo usage rules
- Design system documentation
- Brand consistency checking
