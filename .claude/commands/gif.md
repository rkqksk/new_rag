# /gif - Slack GIF Creator

Create animated GIFs for Slack with various effects and styles.

## Usage
```
/gif create "Ship It!" --effect bounce --color gradient
/gif animate "🎉 Success" --template celebration
/gif text "Loading..." --style pulse --duration 2s
```

## Effects
- bounce, spin, fade, pulse
- shake, wiggle, zoom, morph
- kaleidoscope, explode
- Custom animations
