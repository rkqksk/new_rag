# /art - Algorithmic Art Generation

Generate algorithmic and generative art using mathematical patterns and creative algorithms.

## Usage
```
/art generate fractal --type mandelbrot --colors vibrant
/art create pattern --style geometric --size 2048x2048
/art animate particles --duration 5s --output animation.gif
```

## Algorithms
- Fractals (Mandelbrot, Julia sets)
- Particle systems
- L-systems
- Cellular automata
- Perlin noise patterns
- Geometric tessellations
