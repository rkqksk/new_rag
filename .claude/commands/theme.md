# /theme - Theme Factory

Create and customize visual themes for applications, websites, and documents.

## Usage
```
/theme create dark-mode --primary #007AFF --accent #5856D6
/theme generate corporate --company "TechCorp" --colors brand.json
/theme export vscode --name "Ocean Deep"
```

- Arctic Frost
- Desert Rose
- Midnight Galaxy
- Ocean Depths
