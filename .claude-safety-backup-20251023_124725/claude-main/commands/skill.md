# /skill - Create New Claude Code Skills

Create and package new skills for Claude Code with proper structure and validation.

## Usage
```
/skill create my-new-skill --category development
/skill validate my-skill/
/skill package my-skill/ --output my-skill.zip
```

## Features
- Automatic SKILL.md generation
- Directory structure setup
- Dependency management
- Validation and testing
- Marketplace integration
