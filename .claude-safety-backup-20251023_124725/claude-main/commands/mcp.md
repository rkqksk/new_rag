# /mcp - Model Context Protocol Server Builder

Build MCP servers to extend Claude's capabilities with external tools and services.

## Usage
```
/mcp create github-server --language python
/mcp generate api-wrapper --from openapi.yaml
/mcp test my-server/ --validate
```

## Templates
- Python MCP server
- Node.js MCP server
- REST API wrapper
- Database connector
- File system server
