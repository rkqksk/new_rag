## 🎯 개요

이 문서는 RAG 시스템의 핵심 **3가지 사전**을 종합적으로 정의합니다.

### 3가지 핵심 사전

1. **제품 추천 사전** (50개 항목) - 핵심 🎯
   - 용도, 특성, 크기 기반 추천
   - 신뢰도 0.85-0.95

2. **품질/재질 Q&A 사전** (60개 항목) - 부가 ❓
   - 재질 특성 상세
   - 문제 해결 가이드
   - MSDS/규제 정보

3. **산업용어 사전** (150개 항목) - 지원 🔤
   - 약자 정규화
   - 제품 코드 매핑
   - 기술용어 정의

**총 260개 초기 항목** → 6주 후 410개 (58% 증가)

## 📝 작성 원칙 (5가지)

### 1️⃣ 한국어 우선 (Korean-First)
```
✅ 좋음: "폴리에틸렌 용기는 식품 보관에 적합하고..."
❌ 나쁨: "Polyethylene container is suitable for..."
```
- 모든 설명: **한국어**
- 영어: 기술 용어 (PE, PP) + 제품코드 (PP-50-001)만

### 2️⃣ 구체성 (Specificity)
```
✅ 좋음: "세제 희석액에도 견딜 수 있으며, 60°C까지 안전"
❌ 나쁨: "강하다" 또는 "좋다"
```

### 3️⃣ 재사용성 (Reusability)
한 항목이 여러 질문에서 재사용되어야 함

### 4️⃣ 확장성 (Extensibility)
주당 5-10개씩 점진적 성장 가능

### 5️⃣ 검증 가능성 (Verifiability)
- 제품 스펙 시트
- MSDS
- 산업 표준 (KS, ISO)
- 고객 피드백

## 📋 사전 1: 제품 추천 사전 (50개 항목)

### 구조: 3가지 카테고리

#### A. 용도 기반 (15개 항목)

**예: rec_001 식품 보관**
```yaml
id: "rec_001"
term: "식품 보관용"
synonyms: ["음식 보관", "식용 용기"]
recommended_materials:
  - material: "PE"
    confidence: 0.95
    reason: "FDA 승인, 투명하고 식품 안전"
    code: "PE-CLEAR-500, PE-CLEAR-1L"
key_requirements:
  - "FDA CFR 177 규정 준수"
  - "식품용 인증 필수"
restrictions:
  - "고온 음식 (80°C 이상) - PE 사용 금지"
```

**용도 기반 15개 항목 예상:**
```
식품 저장 (3): 음식, 음료, 냉동
화학약품 (3): 화학약품, 의약품, 화장품
산업용 (3): 샘플, 폐기물, 기타
기타 (6): 추가 용도별
```

#### B. 특성 기반 (20개 항목)

**투명도 (4개)**
- 고투명도 (95%+) → PET
- 중간 (70-85%) → 반투명 PE
- 불투명 → 차광 처리
- 색상 옵션

**내열성 (4개)**
- 극저온 (-40°C)
- 저온 (-20°C)
- 상온 (0-70°C)
- 고온 (80°C+)

**화학 저항성 (4개)**
- 산 저항
- 알칼리 저항
- 유기용제 저항
- 내구성

**경량성 (2개)**
**강도 vs 경량성 (6개)**

#### C. 크기/용량 기반 (15개 항목)

```
소형 (50-250ml):    샘플, 향수, 약
중형 (250-1000ml):  음료, 세제, 식용유
대형 (1-20L):       산업용, 대량 저장
```

## ❓ 사전 2: 품질/재질 Q&A 사전 (60개 항목)

### 구조: 3가지 카테고리

#### A. 재질 특성 (20개)

**PE (폴리에틸렌)**
- 투명도: 85-90%
- 내열성: 50-70°C 권장
- 가격: 저가형 (100%)
- Q&A 예제:
  - "PE는 식품에 안전한가?" → "네, FDA 승인 PE는 안전"
  - "PE가 햇빛에 노출되면?" → "2-3개월 내 갈색 변색"

**PP (폴리프로필렌)**
- 투명도: 75-85%
- 내열성: 80-120°C
- 강도: PE보다 강함

**PET (폴리에틸렌 테레프탈레이트)**
- 투명도: 95-99% (최고)
- 내열성: 60-85°C
- 강도: 매우 강함

**HDPE, PEEK 등 (4개)**

#### B. 품질 문제 해결 (25개)

```
변색 (5):     원인별 해결방안
취약성 (5):   저온/UV 손상
누수 (4):     뚜껑/균열 문제
냄새 (4):     화학용출/저장제품
강도 (3):     파손 방지
```

#### C. MSDS/규제 정보 (15개)

- 식품 저장: 온도 제한, 인증 필수
- 화학약품: 재질별 호환성
- 의약품: 갈색/자주색, 차광 필수
- 재활용: 코드별 분류

```
...
```

**규제/표준 약자 (10개)**
```

ISO:   국제표준기구
KS:    한국산업표준
...
```

**기술 약자 (10개)**
```
UV차단제, 가소제, 항산화제, 윤활제...
```

```
...
```
