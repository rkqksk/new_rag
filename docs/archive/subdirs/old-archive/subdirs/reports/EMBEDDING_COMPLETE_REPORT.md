
### 처리 흐름

1. **데이터 로드**: JSON 파일 파싱 (제품명, 명세, 이미지 정보)
2. **텍스트 구성**: 제품명 + 명세 조합
3. **임베딩 생성**: sentence-transformers로 벡터화
4. **메타데이터**: 제품ID, 카테고리, URL 등 추출
5. **Qdrant 업로드**: 배치 처리로 효율적 저장
