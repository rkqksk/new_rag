# Documentation Archive

This directory contains historical documentation that has been superseded or is no longer actively maintained.

- Old architecture versions
- POC plans
- Architectural redesigns

- Site-specific crawler strategies (onehago, freemold, plasticnet, plastics_kr)

- Crawling status reports
- Recrawl strategies

- Phase 1-5 completion reports

### archive/deployment/

- Preserved old archive directories

## Usage

These documents are kept for:
1. Historical reference
2. Understanding past decisions
3. Learning from previous approaches
4. Context for current architecture

## Accessing Archive

```bash
# Browse specific category
cd archive/crawling

# Search for specific topic
grep -r "keyword" archive/

# View specific document
cat archive/architecture/FINAL_ARCHITECTURE.md
```
