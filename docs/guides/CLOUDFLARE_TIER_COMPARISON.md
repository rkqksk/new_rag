## 🎯 Tier1 - 무료 (시작 추천)

### 비용 구성

**도메인만** (연간):
- `.com`: $9.77/year = **$0.81/month**
- `.dev`: $12/year = **$1/month**
- `.ai`: $120/year = **$10/month**

**Cloudflare 기능**:
- Tunnel: $0 ✅
- HTTPS: $0 ✅
- CDN: $0 ✅
- DDoS: $0 ✅

**총 비용**: **$0.81-10/month**

### 포함 기능

✅ **자동 HTTPS**
- Cloudflare SSL 인증서
- 자동 갱신
- Let's Encrypt보다 빠른 발급

✅ **전 세계 CDN**
- 300+ 위치
- 자동 라우팅
- 기본 캐싱

✅ **무제한 대역폭**
- 트래픽 제한 없음
- 추가 요금 없음

✅ **기본 DDoS 보호**
- Layer 3/4 보호
- 자동 차단

✅ **Cloudflare Tunnel**
- 방화벽 설정 불필요
- 자동 연결 복구
- 무료

### 적합한 경우

- 🎯 **프로토타입/MVP** - 초기 단계
- 🎯 **개인 프로젝트** - 소규모 트래픽
- 🎯 **테스트 환경** - 비용 최소화
- 🎯 **월 10만 요청 이하** - 소규모 사용자

### 제한사항

❌ **엣지 캐싱 없음** - 응답 속도 최적화 제한
❌ **파일 저장소 없음** - S3 대체 불가
❌ **고급 보안 없음** - WAF, Rate Limiting 부재
❌ **이미지 최적화 없음** - 수동 처리 필요

## 💎 Tier2 - 유료 (업그레이드 옵션)

### 비용 구성

**도메인** (연간):
- `.com`: $9.77/year = $0.81/month

**Cloudflare Pro** (월간):
- Pro Plan: $20/month ✅
- Workers: $5/month ✅
- R2 Storage: $5-15/month ✅

**총 비용**: **$30.81-45.81/month**

### 추가 기능

✅ **Cloudflare Workers** ($5/month)
- 엣지 캐싱
- 서버리스 함수
- 10만 요청/day 무료 → 무제한
- 응답 속도: 380ms → 6ms (98% 빠름)

✅ **R2 Storage** ($5-15/month)
- S3 호환 객체 저장소
- 이그레스 비용 $0 (S3는 과금)
- 10GB: $0.15/month
- 100GB: $1.50/month
- 무제한 확장

✅ **Pro Plan** ($20/month)
- 이미지 최적화
- 고급 DDoS 보호
- WAF (웹 방화벽)
- Rate Limiting
- 고급 분석
- 우선 지원

### 적합한 경우

- 🎯 **프로덕션 환경** - 실제 서비스
- 🎯 **월 100만+ 요청** - 대규모 트래픽
- 🎯 **빠른 응답 필요** - 엣지 캐싱
- 🎯 **파일 저장 필요** - 이미지, 비디오
- 🎯 **고급 보안 필요** - WAF, DDoS

### 성능 향상

**캐싱 효과** (Workers):
- 히트율: 75%+
- 응답 시간: 380ms → 6ms
- 오리진 부하: -80%

**파일 저장** (R2):
- S3 대비 이그레스 비용 $0
- 월 1TB 이그레스: S3 $90 vs R2 $0

## 🚀 업그레이드 시기

### Tier1 → Tier2 업그레이드가 필요한 경우

1. **트래픽 증가**
   - 월 100만 요청 이상
   - 동시 접속자 1000명 이상
   - 응답 시간 개선 필요

2. **파일 저장소 필요**
   - 이미지 업로드 기능
   - 비디오 스트리밍
   - 파일 다운로드 제공

3. **보안 강화 필요**
   - DDoS 공격 증가
   - 악성 봇 차단
   - Rate Limiting 필요

4. **비용 절감**
   - S3 이그레스 비용 > $20/month
   - CDN 비용 > $20/month
   - 통합 절감 효과

### 업그레이드 ROI 계산

**예시**: 월 500만 요청, 100GB 파일

**기존 (AWS S3 + CloudFront)**:
- S3 저장: $2.30/month
- S3 이그레스 (100GB): $9/month
- CloudFront (500만 요청): $15/month
- **총**: $26.30/month

**Tier2 (Cloudflare)**:
- R2 저장 (100GB): $1.50/month
- R2 이그레스: $0/month
- Workers (무제한): $5/month
- Pro Plan: $20/month
- **총**: $26.50/month

**차이**: 거의 동일하지만 Cloudflare는 추가 기능 많음

## 🔄 업그레이드 프로세스

### Tier1 → Tier2 전환 (10분)

**Step 1**: Cloudflare Dashboard 로그인
```
https://dash.cloudflare.com
→ 도메인 선택
```

**Step 2**: Pro Plan 활성화
```
Plans → Upgrade to Pro
→ $20/month 확인
→ Upgrade 클릭
```

**Step 3**: Workers 활성화
```
Workers & Pages → Create Application
→ 기존 v11 workers 파일 배포
→ $5/month 확인
```

**Step 4**: R2 활성화 (선택)
```
R2 → Create Bucket
→ Bucket 이름 입력
→ 무료 10GB 확인
```

**완료!** Tier2 활성화됨
