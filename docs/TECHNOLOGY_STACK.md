# Open-Source RAG Technology Stack (2025)

## 🚀 Core Framework
- **RAG Framework**: LangChain
  - Flexibility
  - Extensive integrations
  - Active community support

## 💾 Vector Database
- **Primary**: Qdrant
  - High-performance
  - Rust-based
  - Complex filtering support

## 🌐 Web Framework
- **Backend**: FastAPI
  - High performance
  - Easy integration
  - Async support

## 📊 Relational Database
- **Primary**: PostgreSQL
- **Vector Extension**: pgvector

## 🧪 Testing & Quality
- **Framework**: Pytest
- **Coverage**: 80%+ target
- **Static Analysis**: mypy, black, isort

## 🔍 Additional Tools
- **Embedding**: Sentence Transformers
- **LLM**: Local (Ollama) + OpenAI fallback
- **Monitoring**: Prometheus + Grafana

## 🌈 Guiding Principles
- Open-source first
- Community-driven
- Minimal vendor lock-in
- Maximum flexibility
