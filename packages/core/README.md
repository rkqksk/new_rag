# @rag/core

Core business logic package for RAG Enterprise v10.0.0

## Purpose

Shared business logic and domain models used across multiple applications.

## Usage

```python
from packages.core.models import Product, SearchResult
from packages.core.utils import validate_query
```

## Structure

```
packages/core/
├── __init__.py
├── models/         # Domain models
├── services/       # Business services
├── exceptions/     # Custom exceptions
└── types/          # Type definitions
```
