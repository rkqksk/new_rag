# @rag/utils

Utilities package for RAG Enterprise v10.0.0

## Purpose

Shared utility functions and helpers.

## Usage

```python
from packages.utils.text import clean_text, tokenize
from packages.utils.date import format_date
from packages.utils.validation import validate_email
```

## Structure

```
packages/utils/
├── __init__.py
├── text.py         # Text utilities
├── date.py         # Date/time utilities
├── validation.py   # Validation helpers
├── crypto.py       # Cryptographic utilities
└── logger.py       # Logging helpers
```

## Version

10.0.0 - Unified Maximum
